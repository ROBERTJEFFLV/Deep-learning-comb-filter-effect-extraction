## 1. 方法概述

利用**倒谱分析**检测梳状滤波器效应：

- 麦克风接收 `Y(f) = X(f)·H(f)`, 其中 `H(f) = 1 + A·exp(-j2πfτ)` 是梳状滤波器
- 对数域下梳状调制变为**可加项**，对 log|Y(f)| 做 FFT (倒谱) 在 quefrency = τ 处产生峰值
- 距离 `d = c·τ/2`，其中 c = 343 m/s

**提取特征**: SMD (频谱调制深度), CPR (倒谱峰值比), CPN (归一化倒谱峰值), NDA (帧间差分), CPQ (距离估计)

## 6. 结论与建议

### 最优配置: `freq_800_12000`

- **SMD AUC** = 0.6672
- **最优 TPR** = 76.2%, **FPR** = 49.9%
- **Youden's J** = 0.263
- **阈值** = 0.783

### 数据特征

- pattern=1 比例: 7.56% (极低，与 sim-to-real gap 一致)
- 25cm 以上几乎无可观测 comb pattern

### 距离估计性能

- MAE = 6.81 cm
- 5cm 以内: 41.6%
- 10cm 以内: 77.7%

### 关键发现

1. **真实数据中梳状滤波器效应极难观测**: pattern=1 比例极低，说明在真实户外/实验环境中，反射信号相对于直达声极其微弱
