# 梳状滤波器特征提取管线 v2 — 实验报告

## 1. 背景与问题

### 1.1 原始管线的问题

原始管线的特征提取流程为：`STFT → d1 差分 → 余弦拟合 ω → 距离估计`。

该管线存在以下核心问题：
1. **螺旋桨谐波干扰**：无人机螺旋桨产生的强谐波结构（f₀≈200Hz）在原始幅度谱上完全掩盖了梳状滤波器的微弱调制
2. **余弦拟合不稳定**：在噪声较大的 d1 差分谱上做网格搜索余弦拟合，容易陷入局部最优
3. **缺乏频谱归一化**：没有将梳状滤波器传递函数 H(f) 与声源谱 X(f) 分离的机制

### 1.2 目标

设计新的特征提取管线，使得：
- **有梳状滤波器效应**（有障碍物反射）与**无梳状滤波器效应**（无障碍物）的数据点能被**清晰区分**
- 系统具有**实时性**（单帧处理时间 < 帧周期）
- 对不同距离和反射增益具有**鲁棒性**

## 2. 理论基础

### 2.1 双路径模型

麦克风接收到的信号为直达声与反射声的叠加：

$$Y(f) = X(f) \cdot H(f), \quad H(f) = 1 + A_{rel} \cdot e^{-j2\pi f \tau}$$

其中：
- $X(f)$：声源频谱（螺旋桨噪声）
- $H(f)$：梳状滤波器传递函数
- $A_{rel}$：反射相对增益（0~1）
- $\tau = 2d/c$：往返时延（$c = 343$ m/s）

### 2.2 功率谱的周期调制

$$|H(f)|^2 = 1 + A_{rel}^2 + 2 A_{rel} \cos(2\pi f \tau)$$

在频谱上产生周期为 $\Delta f = 1/\tau$ 的余弦调制。

### 2.3 对数域的可加性

$$\log|Y(f)| = \log|X(f)| + \log|H(f)|$$

对数域中梳状滤波器的调制变为**可加项**，可通过倒谱分析（对 log 谱做 FFT）检测。

### 2.4 倒谱分析的核心原理

对 $\log|Y(f)|$ 做 FFT 得到倒谱（cepstrum）。倒谱中：
- **quefrency bin $n$** 对应频谱中周期为 $P = N_{band}/n$ bins 的调制
- 时延映射：$\tau = n / (N_{band} \cdot \Delta f)$，其中 $\Delta f = SR / N_{FFT}$

### 2.5 关键发现：quefrency 域的分离性

| 信号成分 | 频率特征 | 倒谱 bin 位置 |
|---------|---------|-------------|
| 梳状滤波器 (d=5~55cm) | $\Delta f = 312$~$3436$ Hz | **bins 2~23** |
| 螺旋桨谐波 (f₀=200Hz) | 基频 200Hz | **bin 36** |
| 螺旋桨谐波 (f₀=300Hz) | 基频 300Hz | **bin 24** |

**梳状滤波器（bins 2~23）与螺旋桨谐波（bins 24+）在 quefrency 域天然分离！**

这是新管线能够工作的物理基础——不需要任何谐波抑制，只需在正确的 quefrency 范围内检测峰值。

## 3. 新管线设计

### 3.1 整体架构

```
音频帧 → Hann 窗 → FFT → 截取分析频带 → log 幅度谱
                                              ↓
                         ┌──────────────── 去均值 → FFT (倒谱) → 滑动平均
                         │                                         ↓
                         │                              搜索 comb quefrency 范围
                         │                                    ↓        ↓
                         │                                  CPR      CPN
                         ↓
                   std(去均值后) = SMD
                         ↓
               EMA 归一化 → 帧差分 → NDA
```

### 3.2 提取的特征

每帧提取 5 个特征：

| 特征 | 全称 | 计算方式 | 物理含义 |
|------|------|---------|---------|
| **SMD** | Spectral Modulation Depth | `std(log_band - mean)` | log 频谱的调制深度，有 comb 时 std 更大 |
| **CPR** | Cepstral Peak Ratio | 搜索范围内峰值 / 搜索范围中位数 | 倒谱中 comb 对应 quefrency 的突出程度 |
| **CPN** | Cepstral Peak Normalized | 搜索范围内峰值 / 非 comb 区域均值 | 用非 comb 区域作为基线的归一化指标 |
| **NDA** | Normalized Differential Activity | `mean(\|帧差\|)` | 归一化频谱的帧间变化量 |
| **CPQ** | Cepstral Peak Quefrency | 峰值位置 → τ → d | 用于距离估计 |

### 3.3 倒谱计算的关键实现细节

```python
# ✅ 正确：对 log 谱做标准 FFT 检测周期性
centered = log_band - log_band.mean()
cepstrum = np.abs(np.fft.fft(centered))

# ❌ 错误（v1 的 bug）：irfft 假设输入具有 Hermitian 对称性
# cepstrum = np.abs(np.fft.irfft(centered, n=2*N_band))
```

**这个 bug 是 v1 管线失败的根本原因**。`irfft` 把输入当作半频谱处理，破坏了周期性检测。修复后 CPR 的 AUC 从 0.42（比随机差）提升到 0.94。

### 3.4 最终参数配置

```python
# STFT 参数
N_FFT         = 2048           # 频率分辨率 23.4 Hz
HOP           = 512            # ~10.7ms/帧, ~93 帧/秒

# 分析频带
FREQ_MIN      = 800 Hz         # 下限
FREQ_MAX      = 8000 Hz        # 上限（覆盖 7200Hz 带宽）

# 倒谱搜索范围
TAU_MIN       = 0.25 ms        # 对应 d ≈ 4.3 cm
TAU_MAX       = 4.0 ms         # 对应 d ≈ 68.6 cm

# 时序参数
CEP_AVG       = 4 帧           # 倒谱滑动平均窗口
EMA_ALPHA     = 0.02           # 运行均值 EMA 系数

# 检测阈值
SMD_THRESHOLD = 0.871          # TPR=92%, FPR=0%
```

## 5. 实验结果

### 5.1 实验一：合成数据评估（主实验）

**数据**：200 个无障碍物 clip + 300 个 RIR 合成 clip（反射增益 ≥ 0.3，距离 0~1m）

| CPN | 0.9604 | 3.722 | 91.0% | 10.5% | 0.805 |
| CPR | 0.9380 | 3.065 | 88.0% | 15.0% | 0.730 |
| NDA | 0.7490 | 0.438 | 75.3% | 37.5% | 0.378 |

#### Per-clip AUC-ROC（使用每 clip 的 P95 统计量）

| 特征 | AUC-ROC | TPR | FPR | Youden's J |
|------|---------|-----|-----|-----------|

**结论**：SMD 是最佳单特征，AUC=0.9945，在阈值 0.871 处实现 **TPR=92%、FPR=0%**（零假阳性）。

### 5.3 实验三：距离敏感性分析

## 6. v1 Bug 分析

### 6.1 v1 初始结果（修复前）

| 特征 | AUC-ROC |
|------|---------|
| SMD | ~0.75 |
| CPR | **0.42**（比随机差！） |
| ACP（自相关峰） | ~0.75 |
| NDA | ~0.75 |

### 6.2 根本原因

1. **倒谱计算使用了 `np.fft.irfft` 而非 `np.fft.fft`**
   - `irfft` 假设输入是频谱的前半部分（Hermitian 对称），将其视为频域信号做逆变换
   - 实际上我们需要的是：对 log 幅度谱做**正向 FFT** 来检测频域中的周期性
   - 这是一个根本性的数学错误，导致 CPR 完全失效

2. **ACP（自相关峰）被螺旋桨谐波主导**
   - 自相关域中，螺旋桨谐波（f₀=200Hz → 自相关 lag 8）与梳状滤波器（不同 lag）的映射关系不同于倒谱域
   - 在自相关域中两者**存在重叠**，无法像倒谱域那样干净分离

3. **缺少倒谱帧平均**
   - 单帧倒谱噪声大，需要多帧滑动平均以提高 SNR

### 6.3 修复措施

| 问题 | 修复 |
|------|------|
| `irfft` → `fft` | 使用 `np.fft.fft(centered_log_spectrum)` |
| ACP 被谐波主导 | 替换为 CPN（使用非 comb 区域基线归一化） |
| 单帧噪声大 | 添加 4 帧滑动平均 |
| 去趋势方式 | 简化为减去均值（不再用多项式去趋势） |

## 7. 使用方法

### 7.1 作为 Python 模块

```python
from processing.comb_feature_v2 import CombFeatureConfig, CombFeatureExtractor

# 使用默认最优参数
cfg = CombFeatureConfig()
ext = CombFeatureExtractor(cfg)

# 逐帧处理（实时场景）
frame = audio[start:start + 2048]  # float64, 2048 samples
features = ext.process_frame(frame)
print(f"SMD={features.smd:.4f}  有障碍物: {features.smd > 0.871}")

# 整文件处理（离线场景）
from processing.comb_feature_v2 import extract_features_from_wav
feats, times, dur = extract_features_from_wav("recording.wav")
# feats shape: (N_frames, 6), columns: [smd, cpr, cpn, nda, cpq, dist_est_cm]
```

# 主评估（合成数据）

## 8. 总结

| 维度 | 原始管线 | 新管线 v2 |
|------|---------|----------|
| 方法 | d1 差分 + 余弦拟合 | **倒谱分析 + SMD** |
| 最佳特征 AUC | ~0.75 | **0.9945** |
| 零假阳性时的检出率 | N/A | **92.0%** |
| 真实数据分离度 | 不显著 | **12.89σ** |
| 单帧延迟 | ~1ms | **0.093ms** |
| 实时可行性 | 是 | **是（快 115 倍）** |
| 对弱反射的鲁棒性 | 差 | gain≥0.5 时 TPR≥97.3% |
| 螺旋桨谐波干扰 | 严重 | **倒谱域天然分离** |

新管线的核心优势在于利用了**倒谱域中梳状滤波器调制（bins 2~23）与螺旋桨谐波（bins 24+）天然分离**的物理特性，从根本上解决了谐波干扰问题。SMD 特征在 AUC=0.9945、零假阳性条件下实现 92% 检出率，单帧延迟仅 0.093ms，完全满足实时应用需求。
