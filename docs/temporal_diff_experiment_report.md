# 时序差分实验报告 — 倒谱管线 v2

## 1. 理论分析

### 差分的数学效应

```
ΔL_t(f) = log|Y_t(f)| - log|Y_{t-Δt}(f)|
       ≈ Δlog|X(f)| + A·[cos(2πfτ_t) - cos(2πfτ_{t-Δt})] + ΔN(f)
```

**三个分量的命运**:

| 分量 | 差分后 | 原因 |
|------|--------|------|
| log|X(f)| 声源频谱 | ≈ 0 (静态/慢变) | 螺旋桨 RPM 近似恒定 |
| A·cos(2πfτ) 梳状调制 | 存活 (if τ 变化) / ≈ 0 (if τ 恒定) | 取决于无人机运动状态 |
| N(f) 噪声 | ×√2 (随机噪声) / ↓ (EMA 预平滑后) | 先 EMA 再差分可抑制 |

**关键洞察**: 差分是**双刃剑**
- ✅ **Moving 场景**: τ 持续变化 → 梳状信号存活，声源频谱被消除 → SNR ↑
- ❌ **Static 场景**: τ 恒定 → 梳状信号也被消除 → 检测失效
- 参数 Δt 控制差分的时间尺度：
  - Δt=1 (~21ms): 检测快速变化
  - Δt=10 (~213ms): 检测中等速度运动
  - Δt=40 (~853ms): 检测慢速运动
