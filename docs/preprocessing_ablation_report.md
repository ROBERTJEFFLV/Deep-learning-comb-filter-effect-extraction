## 1. 理论分析

### 1.1 信号模型

接收信号频谱:

```
Y(f) = X(f) · H(f) + N(f)
```

- **X(f)**: 声源（无人机螺旋桨噪声）——强结构化频谱，具有 f₀≈200Hz 的谐波系列
- **H(f)**: 梳状滤波器 H(f) = 1 + A·exp(-j2πfτ)，在真实数据中 A ≪ 1
- **N(f)**: 加性噪声（电噪声、量化噪声、环境噪声）

对数谱:

```
log|Y(f)| ≈ log|X(f)| + A·cos(2πfτ) + noise
```

倒谱 C[n] = |FFT{log|Y(f)| - mean}| 在 quefrency n=τ/(quef_factor) 处产生梳状峰。

**核心挑战**: 真实数据中 A ≪ 1，梳状调制被 log|X(f)| 的自身结构和 N(f) 淹没。

### 1.2 各预处理块的数学分析

#### (a) 噪声门 — Noise Gate

- **原理**: 帧 RMS < 阈值 → 标记为「无梳状效应」(SMD=0)
- **数学效应**: 排除 SNR 极低帧（倒谱被量化/环境噪声主导）
- **对 SMD 的影响**: 消除虚假高 SMD（纯噪声帧的 log 谱方差可能偏高）
- **预测**: ↓ FPR，TPR 几乎不变（comb pattern 只出现在近距离、信号强时）
- **理论判定**: ✅ 安全且有依据

#### (b) 频谱高斯平滑 — Spectral Gaussian Smoothing

- **原理**: 沿频率轴对 log|Y(f)| 做 Gaussian 卷积 (σ bins)
- **数学效应**: 等价于倒谱域乘以 Gaussian 窗 → 衰减高 quefrency 成分
- **对梳状信号的衰减**: exp(-2π²σ_f²τ²)
  - σ=2 bins (≈47Hz), d=10cm (τ=0.58ms): 衰减 = 0.997 → **可忽略**
  - σ=2 bins, d=40cm (τ=2.3ms): 衰减 = 0.95 → **轻微**
  - σ=5 bins, d=40cm: 衰减 = 0.72 → **需注意**
- **对窄带噪声**: 尖峰 δ(f) → Gaussian(σ)，峰值降低 ~σ 倍
- **预测**: σ=1~3 时改善 SNR 而不损失信号；σ>5 可能损失远距离信号
- **理论判定**: ✅ 强依据，σ ≤ 3 安全

#### (c) 预白化 — Pre-whitening (Spectral Envelope Removal)

- **原理**: mag_band / median_filter(mag_band) → 去除频谱包络
- **数学效应**: log 域等价于减去平滑的 log|X(f)| → 剩余 ≈ A·cos(2πfτ) + 残差
- **对 SMD**: 声源频谱形状贡献被去除，SMD 更纯粹反映梳状调制
- **关键参数**: median 窗口大小 W
  - 梳状调制频率 1/τ：d=10cm → Δf≈1724Hz ≈ 74 bins; d=40cm → Δf≈435Hz ≈ 19 bins
  - 螺旋桨谐波间距: f₀≈200Hz ≈ 9 bins
  - **W < 19** 可能平滑掉远距离梳状信号
  - **W ≈ 9-15**: 平滑谐波但保留梳状调制（推荐范围）
- **理论判定**: ✅ 理论可行，W=9~15 为安全范围

#### (d) 时序 EMA 平滑 — Temporal Smoothing

- **原理**: log|Y_t(f)| → α·log|Y_t| + (1-α)·EMA_{t-1}
- **数学效应**: 压制帧间快变噪声（电噪声），保留慢变梳状信号
- **等效时间常数**: τ_ema ≈ 1/α 帧 = (hop/sr)/α 秒
  - α=0.1: τ≈107ms (强平滑); α=0.3: τ≈36ms; α=0.5: τ≈21ms
- **对梳状信号**: 距离变化速度 ~cm/s → 梳状 τ 变化 ~60μs/s → **远慢于** EMA 时间常数
- **预测**: α=0.1~0.3 应改善 SNR
- **理论判定**: ✅ 物理逻辑正确

#### (e) 低频倒谱截除 — Liftering

- **原理**: 将倒谱前 k 个 bin 置零（bin 0 = 均值，bin 1 = 线性趋势 ...）
- **当前已有**: bin 0 已通过 centered = log_band - mean 去除
- **梳状峰位**: bin 2~23 (d=5~55cm)，截除 bin 0~1 **安全**
- **预测**: 对 CPR/CPN 有轻微改善（去除残余趋势干扰），SMD 不受影响
- **理论判定**: ✅ 安全但效果可能有限
