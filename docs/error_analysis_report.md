## 8. 关键发现与改善建议


### 误判根因分析

(由脚本自动识别的模式)


- **static**: SMD p0/p1 分离度仅 0.23σ, 分类本质困难。可能原因: 该角度/运动模式下梳状滤波效应极弱。
- **moving**: SMD p0/p1 分离度仅 0.43σ, 分类本质困难。可能原因: 该角度/运动模式下梳状滤波效应极弱。
- **70deg**: FN 帧平均距离 17.4cm, 远距离处梳状滤波效应减弱是主要漏检原因。
- **80deg**: FN 帧平均距离 19.1cm, 远距离处梳状滤波效应减弱是主要漏检原因。
- **90deg**: FN 帧平均距离 21.0cm, 远距离处梳状滤波效应减弱是主要漏检原因。

### 网络设计改善建议

1. **多通道输入优势**: 新管线的 4 通道 (raw, preprocessed, dt1, abs) 比旧管线更有信息量

2. **能量特征补充**: energy_proxy / snr_proxy 作为辅助输入可帮助区分低信噪比场景

3. **时间上下文**: 长 FP/FN burst 表明帧级分类不足, 需要序列级别的时间建模

4. **距离感知**: FN 集中在远距离处, 模型应学习距离与信号强度的关系
