# 完整管线实验报告 — Macro-Avg AUC 修正版

## 0. Simpson's Paradox 发现

在之前的实验中，`ema_only_0.2` 的 global pooled AUC 最高 (0.6662)，
但 diff 管线在 **每个类别** 上的 AUC 都高于 ema_only。
这是经典的 Simpson's Paradox：

- `ema_only` 在 static/moving 上 FPR≈100%（完全不判别），但因为这些类别的
  pattern=1 帧占总 p1 帧的 ~43%，pooled 时 AUC 被虚高。
- diff 管线真正在做 **判别**，FPR 大幅降低 (20-32% vs 44-100%)。

**修正方案**: 使用 macro-average AUC = mean(per-category AUC) 作为主指标。
